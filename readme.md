# My Gesture Control System

A browser-based hand gesture recognition project using **MediaPipe Hands** and JavaScript.

## Features
- Real-time hand landmark detection using webcam
- Simple gesture classification (open palm, fist, thumbs up)
- Easy to extend for custom actions (e.g., controlling apps, games, or devices)
